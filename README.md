# Installation 
- clone this project to your local 
```bash
git clone https://github.com/thoriqabdul/DBO.git
```
- run go get
```bash
go get
```

- setup .env
```bash
cp .env.example .env
```
customize the values in the environment

# RUN PROJECT
- with docker
```bash
docker-compose up -d
```
-local
```bash
go run main.go
```
